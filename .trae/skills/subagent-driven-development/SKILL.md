# Subagent-Driven Development

Parallel task implementation with quality checkpoints.

## When to Use

- Complex features with independent components
- Multiple tasks that can run in parallel
- Large implementation that needs to be broken down

## Process

1. Create implementation plan
2. Break down into parallel tasks
3. Each task follows: Implement → Spec Review → Code Quality Review
4. Integrate and verify
