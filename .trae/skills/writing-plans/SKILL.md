# Writing Plans

Transform designs into actionable implementation plans.

## When to Use

- After `/plan-eng-review` or design is approved
- User wants to create an implementation plan
- Moving from design to execution phase

## Process

1. Review approved design
2. Break down into implementation tasks
3. Identify dependencies
4. Define acceptance criteria
5. Create actionable task list
