# QA Testing

Systematic QA testing workflow using real browser.

## When to Use

- User asks to "test this"
- Feature needs verification
- Pre-deployment check
- Bug fix verification

## Process

1. Identify test scope and critical paths
2. Execute tests using `/browse` or Playwright
3. Document any issues found
4. Fix issues if possible, or report
